# DNA Sequence Classification using Machine Learning

## Overview
This project demonstrates classification of DNA sequences using k-mer based feature extraction and machine learning models.

## Methods
- k-mer encoding (k=3)
- Logistic Regression
- Support Vector Machine (SVM)

## Tools
- Python
- scikit-learn

## How to Run
```bash
pip install -r requirements.txt
python main.py
```
